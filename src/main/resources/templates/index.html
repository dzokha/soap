Hướng dẫn sử dụng Pipeline

  1. Kiểm soát chất lượng (QC): Upload file `.fastq.gz` để kiểm tra.
  2. Làm sạch (Trimming): Cắt bỏ đầu mồi và base lỗi.
  3. Lắp ráp (Assembly): Tạo bộ gene vòng (Circular genome).
  4. Chú giải (Annotation): Phân tích và dán nhãn các gene.
  5. Trực quan hóa: Xem bản đồ gene trực tiếp bằng SeqViz.
