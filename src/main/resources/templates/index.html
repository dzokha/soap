Pharokka Annotation Engine

Đầu vào của quá trình này là file Fasta (.fasta) đã được lắp ráp từ bước Assembly. Kết quả sẽ được xuất ra định dạng GenBank (.gbk) chuẩn NCBI.

Hướng dẫn sử dụng Pipeline

Hệ thống SOAP được thiết kế theo quy trình chuẩn cho phân tích Phage:

  1. Kiểm soát chất lượng (QC): Upload file `.fastq.gz` để FastQC kiểm tra độ dài, GC content và Adapter.
  2. Làm sạch (Trimming): Trim Galore sẽ cắt bỏ đầu mồi và base lỗi.
  3. Lắp ráp (Assembly): Chạy Unicycler để tạo bộ gene vòng (Circular genome).
  4. Chú giải (Annotation): Pharokka phân tích và dán nhãn các gene.
  5. Xuất dữ liệu: Tải file .gbk để xem bản đồ trên SnapGene.
